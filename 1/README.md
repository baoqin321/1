# 去水印工作台

一个纯前端的本地去水印页面，直接打开 `index.html` 即可使用。

## 功能

- 图片去水印：上传后拖拽框选区域，支持邻域填补、模糊、像素化、颜色覆盖。
- 视频去水印：上传后拖拽框选区域，浏览器本地逐帧预览处理，并导出为 `WebM`。
- 无后端：默认不上传文件，所有处理都在本地浏览器执行。

## 使用方式

1. 打开 `index.html`。
2. 选择“图片”或“视频”。
3. 上传文件后，在预览区域拖拽框选水印。
4. 选择处理方式和强度。
5. 图片点击“应用到图片”并下载，视频点击“导出 WebM”。

## 说明

- `邻域填补` 适合边角、面积较小的水印，复杂背景未必能完全自然修复。
- 视频导出依赖浏览器的 `MediaRecorder` 能力，推荐使用较新的 Chromium 内核浏览器。
- 部分浏览器在视频导出时可能无法保留原音轨。
